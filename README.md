## 💻 Frontend: Real-Time Chat Interface

A responsive, high-performance web interface built for a seamless user experience.

### 🛠 Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **State Management:** Context API / Redux
- **Client-Side Sockets:** Socket.io-client

### 🚀 Deployment
- **Hosting:** Managed by **Vercel** for global edge delivery.
- **Communication:** Connects to the secure backend at `https://api.talha2424.me`.

### 🔑 Environment Variables
- `NEXT_PUBLIC_API_URL=https://api.talha2424.me`
